# 基础后台管理系统模板

一个基于 `Vue 3`、`Vite`、`TypeScript`、`Element Plus` 构建的后台管理系统基础模板，适合作为中后台项目的起步框架进行二次开发。

## 项目地址

- GitHub：`https://github.com/LingWei888/VUE3-Frame.git`

## 项目简介

本项目已整理为更纯净的后台管理模板，保留了后台系统开发常用的基础能力，适合用于：

- 企业后台管理系统
- 运营管理平台
- 内部业务管理平台
- 通用中后台前端基础框架

当前版本重点提供：

- 完整的基础布局结构
- 路由与菜单组织能力
- Vuex 状态管理
- 统一请求封装
- 国际化配置能力
- 常见业务页面示例
- 可扩展的主题机制
- 统一的右侧抽屉表单交互

## 技术栈

- `Vue 3`
- `Vite`
- `TypeScript`
- `Element Plus`
- `Vue Router 4`
- `Vuex 4`
- `Axios`

## 目录说明

```text
src/
├─ api/               接口管理
├─ assets/            静态资源与公共样式
├─ components/        公共组件
├─ config/            项目配置
├─ directive/         自定义指令
├─ layout/            系统布局
├─ locale/            国际化配置
├─ router/            路由管理
├─ store/             状态管理
├─ theme/             主题配置
├─ utils/             工具函数
└─ views/             页面模块
```

## 功能特性

- 基础后台布局

## 开发说明

### 1. 克隆项目

```bash
git clone https://github.com/LingWei888/VUE3-Frame.git
```

### 2. 安装依赖

```bash
npm install --legacy-peer-deps
```

### 3. 启动开发环境

```bash
npm run dev
```

### 4. 生产打包

```bash
npm run build
```
